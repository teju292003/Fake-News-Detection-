# AI Fake News Detector 🕵️‍♀️📰

A high-accuracy Machine Learning web application that detects whether a given news article or headline is REAL or FAKE.

## 🚀 Overview
This project uses Natural Language Processing (NLP) to analyze the text of news articles. The backend is built with **Django**, and the machine learning model is trained using a **Passive-Aggressive Classifier** and a **TF-IDF Vectorizer**, achieving over 91% accuracy.

## 🛠️ Tech Stack
* **Frontend:** HTML, CSS (Bootstrap 5)
* **Backend:** Python, Django
* **Machine Learning:** Scikit-Learn, Pandas, NLTK, Joblib
* **Algorithm:** TF-IDF Vectorizer + Passive Aggressive Classifier
